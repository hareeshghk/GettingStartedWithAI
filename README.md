# GettingStartedWithAI

A practical exploration of AI-powered development using GitHub Copilot and Visual Studio Code.

## Project Overview

This repository demonstrates how AI can accelerate web development by building a complete **Task Manager Web Application** using only AI assistance in Visual Studio Code.

## 🎯 Task Manager Application

### What Was Built
A fully-featured, accessible task manager with:
- ✅ Add, delete, and complete tasks
- 🎨 Modern, responsive design with dark/light mode
- 🔍 Filter tasks (All, Active, Completed)
- ♿ Full accessibility support (ARIA, keyboard navigation)
- 💾 Persistent storage using localStorage
- 📱 Mobile-responsive layout
- ⚡ Real-time toast notifications

### Tech Stack
- **HTML5** - Semantic markup with accessibility features
- **CSS3** - Modern styling with CSS Grid, Flexbox, and CSS custom properties
- **Vanilla JavaScript** - ES6+ features, no frameworks needed
- **Progressive Enhancement** - Works without JavaScript enabled

### Key Features Developed with AI

#### 🤖 AI-Driven Development Process
- **Complete file generation** - HTML structure, CSS styling, and JavaScript functionality
- **Accessibility implementation** - ARIA attributes, screen reader support, keyboard shortcuts
- **Responsive design** - Mobile-first approach with flexible layouts
- **Theme system** - Dark mode toggle with system preference detection
- **Error handling** - Robust localStorage operations with fallbacks

#### 🎨 Design Excellence
- **CSS Custom Properties** for consistent theming
- **Smooth animations** with respect for `prefers-reduced-motion`
- **Focus management** for keyboard users
- **Visual feedback** through toast notifications and state changes

#### 🛡️ Security & Performance
- **XSS Prevention** through proper HTML escaping
- **Efficient DOM manipulation** with event delegation
- **Optimized CSS** with minimal specificity conflicts
- **Performance considerations** for scalable task lists

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/hareeshghk/GettingStartedWithAI.git
   cd GettingStartedWithAI
   ```

2. **Open in your browser**
   - Simply open `index.html` in any modern web browser
   - No build process or server required!

3. **Start managing tasks**
   - Add tasks using the input field
   - Toggle completion with checkboxes
   - Filter by status using the toolbar
   - Switch themes with the header toggle
